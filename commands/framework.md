---
description: Apply or derive a mental model / framework for a problem.
argument-hint: <problem or decision>
---
Give a useful framework for: $ARGUMENTS

- If a well-known model fits, name it, explain it briefly, and apply it
  concretely to this case.
- If none fits cleanly, derive a simple framework (the few dimensions/steps that
  actually matter) and apply it.
- End with what the framework reveals that wasn't obvious.
Prefer one sharp framework applied well over a list of names.
