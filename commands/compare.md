Compare: $ARGUMENTS

Identify the dimensions that actually matter for choosing between them, compare
each option across those dimensions (a table is fine), call out where they
genuinely differ vs. where it's a wash, and finish with "pick X if..., pick Y
if..." guidance. Avoid false balance - if one is clearly better for most cases,
say so.
